# 함정 추론  (reasoning)

**문제**
> 야구 방망이와 공이 합쳐서 1,100원이다. 방망이는 공보다 1,000원 비싸다. 공의 가격은? 정답과 풀이 한 줄.

## 🅰️  GPT-5.2
공의 가격을 x원이라 하면 방망이는 (x + 1,000)원이고, x + (x + 1,000) = 1,100을 풀면 x = 50이므로 **공은 50원**입니다.

## 🅱️  Claude-Opus-4.5
**정답: 50원**

**풀이:** 공을 x원으로 두면 방망이는 x+1000원이므로 x+(x+1000)=1100 → 2x=100 → x=50원
