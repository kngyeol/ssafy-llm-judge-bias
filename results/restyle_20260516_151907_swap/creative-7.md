# 역설적 책 제목  (creative)

**문제**
> '성공하려면 실패하라' 같은 역설적 자기계발서 제목 8개를 작성하라. 각 20자 이내, 클리셰 피하기. 번호 매기기, 부연설명 금지.

## 🅰️  GPT-5.2
1. 느려야 앞서간다
2. 포기가 너를 멀리 데려간다
3. 비움이 채움이다
4. 지는 연습
5. 멈춤이 방향을 만든다
6. 약해져야 강해진다
7. 쉬는 자가 길을 연다
8. 잊어야 기억된다

## 🅱️  Claude-Opus-4.5

2. 내려놔야 붙잡힌다

4. 의심해야 확신이 선다  
5. 놓아야 오래 남는다  
6. 멈춰야 비로소 간다  
7. 비워야 채워진다
