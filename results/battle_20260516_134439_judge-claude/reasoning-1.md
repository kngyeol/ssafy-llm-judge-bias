# 논리 퍼즐  (reasoning)

**문제**
> 5명(A,B,C,D,E)이 일렬로 앉는다. 조건: A는 B 바로 옆에 있다. C는 양 끝자리가 아니다. D는 A보다 왼쪽이다. E는 정중앙(3번째)이다. 가능한 모든 배열을 나열하고 개수를 답하라. 풀이 과정 간단히 포함.

## 🅰️  GPT-5.2
사람 5명 A,B,C,D,E를 1~5번 자리에 앉힌다.

조건 정리  
- E는 3번(정중앙) 고정  
- C는 양끝(1,5) 불가 → C는 2번 또는 4번  
- A는 B와 이웃(연속 자리)  
- D는 A의 왼쪽

이제 E=3을 고정하고 남은 자리 {1,2,4,5}에 A,B,C,D를 배치한다.

---

## 1) C=2인 경우
남은 자리: {1,4,5}에 A,B,D 배치.  
A와 B는 이웃이어야 하는데, 가능한 이웃쌍은 (4,5)뿐이다.  
따라서 A,B가 4-5를 차지.

- (A,B)=(4,5)면 D는 1인데 D가 A(4)보다 왼쪽 → 가능  
- (A,B)=(5,4)면 D는 1인데 D가 A(5)보다 왼쪽 → 가능

배열(1~5):
1) D C E A B  = D C E A B  
2) D C E B A  = D C E B A  

→ **2가지**

---

## 2) C=4인 경우
남은 자리: {1,2,5}에 A,B,D 배치.  
이웃쌍은 (1,2)뿐이므로 A,B가 1-2를 차지하고 D는 5에 간다.

- (A,B)=(1,2)이면 D=5인데 D가 A(1)보다 왼쪽이어야 함 → 5는 오른쪽, 불가  
- (A,B)=(2,1)이면 D=5인데 D가 A(2)보다 왼쪽이어야 함 → 불가

→ **0가지**

---

# 가능한 모든 배열과 개수
가능한 배열(왼→오):
- **D C E A B**
- **D C E B A**

따라서 **총 개수 = 2가지**.

## 🅱️  Claude-Opus-4.5
# 5명 일렬 배열 문제 풀이

## 조건 정리
1. **A-B 인접**: A와 B는 바로 옆에 붙어있음
2. **C 위치**: 양 끝(1번, 5번) 불가 → 2번 or 4번만 가능
3. **D < A**: D는 A보다 왼쪽에 위치
4. **E 위치**: 정중앙(3번) 고정

## 풀이

### 고정 요소
```
위치:  1 - 2 - 3 - 4 - 5
              E (고정)
```

### A-B 블록 배치 가능 경우

E가 3번에 고정되어 있으므로, A-B 인접 블록이 들어갈 수 있는 위치쌍:
- **(1,2), (4,5)** ✓ (3번을 포함하지 않음)

C는 2번 또는 4번만 가능하므로, A-B 블록 위치와 C 위치의 조합을 분석:

---

### Case 1: A-B가 (1,2)에 위치

C는 2번 불가 → **C는 4번**
D는 5번 (남은 자리)

**D < A 조건 확인:**
- AB 순서 (1,2): A=1위치 → D(5) < A(1)? ❌
- BA 순서 (1,2): A=2위치 → D(5) < A(2)? ❌

→ **Case 1: 불가능**

---

### Case 2: A-B가 (4,5)에 위치

C는 4번 불가 → **C는 2번**
D는 1번 (남은 자리)

```
위치:  1 - 2 - 3 - 4 - 5
       D   C   E  [A-B 블록]
```

**D < A 조건 확인:**
- AB 순서 (4,5): A=4위치 → D(1) < A(4)? ✓
- BA 순서 (4,5): A=5위치 → D(1) < A(5)? ✓

→ **2가지 배열 가능**

---

## 가능한 모든 배열

| 번호 | 1번 | 2번 | 3번 | 4번 | 5번 |
|:---:|:---:|:---:|:---:|:---:|:---:|
| **1** | D | C | E | A | B |
| **2** | D | C | E | B | A |

---

## 최종 답: **2개**

## ⚖️  심판 (CLAUDE_SONNET_4_5)
WINNER: TIE

REASON: 두 답변 모두 정확히 2개의 배열(D C E A B, D C E B A)을 도출했고, 풀이 논리도 동일하게 올바르다. A는 C=2/C=4로 경우를 나누어 간결하게 설명했고, B는 A-B 블록 중심으로 표를 활용해 시각적으로 명확하게 제시했다. 접근 방식만 다를 뿐 정확성과 완성도는 동등하다.
