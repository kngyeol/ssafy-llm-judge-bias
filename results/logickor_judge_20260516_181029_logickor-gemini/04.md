A: '범인은 C에요.'

C: '저는 범인이 아니에요.''

### 🅰️ GPT-5.2
세 사람 중 **한 명만 거짓말**, **두 명은 진실**이라고 했습니다.

- A: “범인은 C”
- B: “제가 범인”
- C: “저는 범인이 아니다”

가능한 범인을 하나씩 가정해 보면:

1) **C가 범인**이면  
- A의 말(범인은 C) = **참**  
- B의 말(내가 범인) = **거짓**  
- C의 말(나는 범인 아님) = **거짓**  
→ 거짓말이 2명이 되어 조건(거짓 1명)과 **불일치**

2) **B가 범인**이면  
- A의 말(범인은 C) = **거짓**  
- B의 말(내가 범인) = **참**  
- C의 말(나는 범인 아님) = **참**  
→ 거짓말이 정확히 1명(A)이고 나머지 2명은 참 → **조건과 일치**

3) **A가 범인**이면  
- A의 말(범인은 C) = **거짓**  
- B의 말(내가 범인) = **거짓**  
- C의 말(나는 범인 아님) = **참**  
→ 거짓말이 2명이 되어 **불일치**

따라서 범인은 **B**입니다.

이런 유형(‘한 명만 거짓말’, ‘서로를 지목’, ‘자기부정/자기고백’)의 논리 퍼즐은 조건이 단순한 대신 **경우를 나눠 모순을 찾는 과정**이 깔끔해서, 논리적 사고 훈련용으로 좋다고 봅니다. 다만 문장들이 너무 단순하면(예: “C가 범인”, “내가 범인”, “난 아님”)는 경우의 수가 빨리 정리되어 난이도가 낮아질 수 있어요. 재미를 높이려면 “둘 중 하나는 참” 같은 **복합 진술(OR/AND)**을 섞으면 좋아집니다.

아래에 비슷한 문제를 하나 만들어서 풀이까지 제시하겠습니다.

---

## 문제
A, B, C 중 한 명이 도난 사건의 범인이다.  
세 사람 중 **정확히 한 명만 거짓말**을 하고, 나머지 **두 명은 진실**을 말한다.

- A: “범인은 B가 아니야.”
- B: “범인은 C야.”
- C: “A와 B의 말 중 **정확히 하나만** 참이야.”

범인은 누구인가?

---

## 풀이
핵심은 C의 발언:  
> “A와 B의 말 중 정확히 하나만 참”

즉, **A와 B는 ‘하나는 참, 하나는 거짓’**이어야 합니다.

이제 범인을 가정해 보겠습니다.

### 1) 범인이 B인 경우
- A: “범인은 B가 아니야” → **거짓**
- B: “범인은 C야” → B가 범인이므로 **거짓**
- 그러면 A와 B의 말은 둘 다 거짓이므로  
  C의 말(정확히 하나만 참)도 **거짓**  
→ 거짓말이 A,B,C 총 3명 → 조건 위반

### 2) 범인이 C인 경우
- A: “범인은 B가 아니야” → C가 범인이니 B는 범인 아님 → **참**
- B: “범인은 C야” → **참**
- 그러면 A와 B는 둘 다 참이므로  
  C의 말(정확히 하나만 참) → **거짓**
→ 거짓말이 C 한 명뿐 → 조건(정확히 1명 거짓)과 일치  
✅ 이 경우는 가능

### 3) 범인이 A인 경우
- A: “범인은 B가 아니야” → A가 범인이니 B는 범인 아님 → **참**
- B: “범인은 C야” → **거짓**
- A와 B는 하나 참/하나 거짓이므로  
  C의 말(정확히 하나만 참) → **참**
→ 거짓말이 B 한 명뿐 → 조건과 일치  
✅ 이 경우도 가능

여기서 범인이 **A 또는 C 둘 다 가능**해져서, 문제 조건만으로는 범인을 하나로 확정할 수 없습니다.  
즉, 이 퍼즐은 “해답이 유일하지 않다”는 점이 드러납니다.

---

## (보너스) 유일해지도록 한 문장만 수정
위 퍼즐을 유일해로 만들고 싶으면 C의 진술을 이렇게 바꾸면 됩니다:

- C: “A와 B의 말은 **둘 다 참이야**.”

그럼 다시 보면,
- 범인이 C일 때: A 참, B 참 → C 참 (거짓 0명)이라서 불가가 아니라? 잠깐, C가 범인이면 C의 말(둘 다 참)은 참이므로 거짓말 0명이 되어 조건 위반.
- 범인이 A일 때: A 참, B 거짓 → C 거짓 (거짓 2명) 조건 위반.
- 범인이 B일 때: A 거짓, B 거짓 → C 거짓 (거짓 3명) 조건 위반.

이렇게 바꾸면 오히려 “해가 없음”이 되어버리니, 대신 아래처럼 수정하면 유일해가 됩니다:

- C: “A와 B의 말 중 **둘 다 참은 아니다**.”(즉, 적어도 하나는 거짓)

이 경우를 체크하면:
- 범인 A: A 참, B 거짓 → C 참 (거짓 1명: B) 가능
- 범인 C: A 참, B 참 → C 거짓 (거짓 1명: C) 가능  
또 2개가 남으니, 더 강한 조건이 필요합니다.
